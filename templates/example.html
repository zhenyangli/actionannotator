Example

This page demonstrates how to annotate step by step.

1) Check the target object shown in mutiple video frames.
2) Watch the video and provide a description about what the target object is doing in the video:
Describe in ONE sentence, explicitly specifying the CATEGORY of the object (like car, dog, etc.) and the ACTION that the object is doing (like jumping, walking, etc.).
Other properties (like COLOR or LOCATION) of the object may also be used in the description so that the target object can be IDENTIFIED UNIQUELY from the video.
